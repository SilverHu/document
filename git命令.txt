
--------------------------------------------实用指令------------------------------------------------
生成ssh rsa
	ssh-keygen -t rsa -C "邮箱"
	
配置多个rsa
	需要在默认的.ssh文件夹下创建config文件，添加
	Host github.com
    	HostName        github.com
    	User            SilverHu
    	IdentityFile    /d/git/id_rsa_SilverHu
	
	将.public的内容复制到git的ssh key中
	
验证ssh是否配置成功
	ssh -T git@github.com
	成功结果：Hi SilverHu! You've successfully authenticated, but GitHub does not provide shell access.

添加项目到git
	git init
	git remote add origin git@github.com:yourName/yourRepo.git

设置忽略文件
	在项目底下配置.gitignore文件，在文件中设置忽略路径
	
查看分支
	git branch -a
	
强制检出/切换分支或分支指定文件
	git checkout <branch> [file] [-f]

新增文件加入git版本库中
	git add 文件名/-A

删除git版本库中文件
	git rm 文件名
	git rm -r 文件夹名

暂存并提交，只会提交已经在git仓库的内容
	git commit -a -m "提交的描述信息"

查看已合并|未合并的本地仓库分支
	git branch [--merged | --no-merged]

将其他分支合并到本分支
	git merge <branch name>

合并冲突，解决冲突后
	git add 文件名	标记冲突已解决
	git commit	提交
	
删除远程分支
	git push origin :branch-name
	
推送本地仓库代码到远程仓库
	git push [remote_branch] [local_branch]

取消暂存
	git reset [file]

恢复文件
	git checkout -- [file]

查看本地仓库分支
	git branch

查看本地仓库分支最后一次提交情况
	git branch -v


查看远程仓库名
	git remote -v

新建分支並立即切換到新分支
	git checkout -b <branch_name>
修改分支名称
	git branch -m <new_name> 

从远程仓库拉取代码到本地仓库，并建立跟踪关系
	git clone http://xxx.git
	get clone git@xxx.git
